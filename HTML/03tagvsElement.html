Tag vs Element

Tag

They are come in between < Angularbrakets> like h1, p, div, span, etc

90% tags are come with opening and closing tag but some tags are self closing tag like br, hr, img, input, etc

Self closing tags are also know as void tag or empty tag.

Element

Star tag - some content - End tag

Attributes

Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.
